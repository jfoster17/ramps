ramps
=====

Code for RAMPS (the Radio Ammonia Mid-Plane Survey)

GBT13B-312

make_pilot_cat_and_scripts.py automatically generates
tiles and stripes for the pilot program, including catalogs 
for each field.

snake_obs.py, vegas_config.py and ramps_catalog.txt will be 
used to test VEGAS with the KFPA on a known field (G11.11-0.12).
